# TREXIMA - SuccessFactors Translations Export and Import Management Accelerator
Function 1: Exports translations from SF and generates a workbook.
Function 2: Once the translations have been verified and updated in the workbook, this tool processes the data model sheets and generates updated data model files for importing the translations changes to SF.

# Installation
1. Make sure that python is installed in the system or virtual machine where TREXIMA is going to run
2. Make sure the python installation path is added to the system's environment variable. 
4. Download the repository contents which includes requirements.txt file
5. Install all libraries using following pip command

  pip install -r "full path to the requirements.txt file downloaded"

6. Change the '.py' file extension to '.pyw' and double click the python program having 'pyw' file extension to run it
